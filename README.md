# ManboML: A Manboster Project

Manboster aims to make inferences in purego, but there is none to do this. So here is ManboML, a manboster project using purego to drive calculations in model inferences.

Due to the architecture limitation of ffi plugin, this will help more minor devices to run models internally.

## Features

- [ ] Pure Golang, no dependencies
- [ ] Support tensors and algorithms
- [ ] GGUF file support
